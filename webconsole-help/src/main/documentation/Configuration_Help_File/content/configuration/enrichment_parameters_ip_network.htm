IP network enrichment

The IP network enrichment is an algorithmic enrichment. For a given IP address, It generates location data in JSON format, including:

  • Country
  • Region (e.g. State)
  • City
  • Postal Code
  • Latitude and Longitude
  • Metro Code
  • Area Code
